# [Android-UML-Diagram](https://github.com/JantHsueh/Android-UML-Diagram)

android 部分源码的流程图、时序图、类图，帮助更好的理解和记忆源码。

项目中包括，流程图、时序图、类图、辅助理解的图片以及UML图源文件（可导入画图工具）。并不是每个源码分析都包含这些，例如：函数的分析没有类图

- 画图工具：https://www.processon.com
